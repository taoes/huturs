pub fn page_rainbow(

) -> Vec<i32>
Expand description

生成分页彩虹条页码数组

§参数

  • page_no - 当前页码（从1开始）
  • total_page - 总页数
  • display_count - 显示的页码数量

§返回值

返回要显示的页码数组

§示例

use huturs_core::util::page_rainbow;
let result = page_rainbow(5, 20, 6);
assert_eq!(result, vec![3, 4, 5, 6, 7, 8]);
